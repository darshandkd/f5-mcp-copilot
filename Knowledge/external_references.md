# F5 External Documentation References

## Official Documentation

### AskF5 Knowledge Base
- **URL:** https://my.f5.com/manage/s/article
- **Use for:** K-articles, troubleshooting, known issues
- **Search:** `site:my.f5.com/manage/s/article [topic]`

### CloudDocs (API/Automation)
- **URL:** https://clouddocs.f5.com
- **Use for:** AS3, DO, TS, iControl REST API
- **Key pages:**
  - AS3: https://clouddocs.f5.com/products/extensions/f5-appsvcs-extension/latest/
  - DO: https://clouddocs.f5.com/products/extensions/f5-declarative-onboarding/latest/
  - TS: https://clouddocs.f5.com/products/extensions/f5-telemetry-streaming/latest/
  - iControl REST: https://clouddocs.f5.com/api/icontrol-rest/

### TMSH Reference
- **URL:** https://clouddocs.f5.com/cli/tmsh-reference/latest/
- **Use for:** Command syntax, parameters, examples

### F5 DevCentral (Community)
- **URL:** https://community.f5.com
- **Use for:** iRules, code samples, community solutions

## Search Patterns

When local knowledge is insufficient, search:
```
site:my.f5.com [error message or topic]
site:clouddocs.f5.com [API or automation topic]
site:community.f5.com [iRule or code example]
```
